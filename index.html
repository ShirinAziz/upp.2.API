Routs

  • GET /users - Skickar tillbaka all befintlig data.

  • GET /users/{id} - Skickar tillbaka sepcifik data med angivet id eller status 204 om den efterfrågade data saknas.

  • POST /users - Skapar en ny användare med JSON objekt på formatet {"username": "unikt namn", "password": "", "name": "", email": "",}. Alla fält är obligatoriska! Fyll i username, password, name och email, annars skickas 422-meddelande (unprocessable entity).
